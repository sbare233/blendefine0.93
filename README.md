# 混元灵印 / Blendefine

> **在道法机关之间，将阴阳五行炼化为天地万象。**
>
> *一款关于炼丹炉、龙爪、和时序编程的东方玄幻解谜游戏。*

---

## 概览

《混元灵印》是一款受 Opus Magnum 启发的"机关编程"解谜游戏。你扮演一位天师，在灵盘上搭建由**青龙爪**、**朱雀炉**、**五脏凝练炉**、**混元器**组成的道法流水线，通过编排**时序指令**，将基础的**阴阳五行**灵印一步步合成为四象、八卦、天干地支、八仙、九龙子等更高等的灵印。

每道关卡要求你在有限的格子和材料约束下，设计出高效的合成方案——像写程序一样编排指令，像炼丹一样转化元素。

---

## 核心机制

### 机关 / 机器

| 机关 | 作用 |
|------|------|
| **青龙爪** | 抓取、搬运、旋转灵印。你的"机械臂" |
| **朱雀炉** | 连接不相生的灵印，使它们结合成宝石组 |
| **混元器** | 用阴阳之力将灵印转化为相生序列中的下一个 |
| **五脏凝练炉** | 按顺时针顺序放入最多五种材料，合成高级灵印 |
| **玄武路** | 用水流运输龙爪，控制流动方向 |
| **六合周天混元阵** | 巨型阵法，直接在原位转换整条相生链 |
| **灵印供应器** | 从灵盘中取出基础材料 |

### 时序指令

就像编程一样编排龙爪和行为：

| 指令 | 键位 | 功能 |
|------|------|------|
| **取** | Q | 龙爪抓取灵印 |
| **围** | W | 设置龙爪的目标位置和角度 |
| **逸** | E | 龙爪放下灵印 |
| **安** | A | 停止机器 |
| **始** | S | 启动机器 |
| **递** | D | 循环执行框选的指令序列 |
| **纵** | Z | 玄武路朝黑色方向流动 |
| **蓄** | X | 玄武路朝白色方向流动 |
| **抄** | C | 复制框选的指令 |

### 灵印合成体系

从七种基础元素出发，构建出整个东方玄幻的符号宇宙：

```
阴阳 → 五行（金木水火土）→ 四象 → 八卦
                                     → 天干地支 → 八仙
                                                  → 七星 → 九龙子
```

合成规则基于**五行相生相克**和**阴阳转化**，形成一个深度嵌套的合成树。游戏内置的**灵符盒**（PuzzleSolver）可以可视化查看任意灵印的最短合成路径。

### 关卡

- 80+ 道关卡，从基础教程到复杂的多目标合成
- 每关有特定的目标灵印、可用材料和格子限制
- 通关推进剧情线，解锁新机关和合成配方
- 支持存档/读档
